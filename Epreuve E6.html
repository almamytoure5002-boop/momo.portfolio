Situation N°1

Administration Système

RP1 : mise en place d'un annuaire ldap sur Windows afin de centraliser l'active directory

Windows Server 2022 GPO & Sécurité NTFS

Cette mission visait à mettre un terme à la dispersion des fichiers audio (jingles, playlists) stockés localement sur les postes de l'équipe Animation. J'ai pris en charge le déploiement complet d'un Active Directory (AD DS) sous Windows Server 2022. Cela a impliqué la création d'une Unité Organisationnelle dédiée, la gestion rigoureuse des habilitations via des groupes de sécurité et la configuration de droits NTFS restrictifs. Pour optimiser l'expérience utilisateur, j'ai élaboré une GPO de mappage automatisant la connexion à un lecteur réseau partagé.

Situation N°2

Gestion de Parc (ITSM)

RP2 : mise ne place d'une solution de gestion de ticket glpi

GLPI v10 Debian 12 Linux Liaison LDAP/AD

Avant cette intervention, les incidents chez Stadium Company manquaient de traçabilité. J'ai donc déployé un serveur LAMP sous Debian 12 pour héberger la solution de helpdesk GLPI. Le cœur technique de cette réalisation fut l'intégration de l'authentification centralisée LDAP. En connectant GLPI à l'Active Directory, j'ai permis aux utilisateurs de s'authentifier avec leurs identifiants de session habituels tout en automatisant l'importation des comptes.
